# Support

## Recommended Support Path

For product questions, installation help, commercial usage, or direct contact, use:

- Website: https://aramin.co
- GitHub Issues: https://github.com/ARaminco/aramis/issues

## Good Bug Reports Include

- Aramis version.
- Operating system and CPU architecture.
- Desktop app or web/server mode.
- Node.js version when running from source.
- AI provider and model, without exposing API keys.
- Exact reproduction steps.
- Relevant logs with secrets removed.

## Security Reports

Do not report security vulnerabilities in public issues. Follow [SECURITY.md](SECURITY.md).
